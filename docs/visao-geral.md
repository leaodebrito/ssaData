# Visão Geral

## O que é o SSA Dados?

O **SSA Dados** é uma plataforma web interativa para análise e visualização de dados urbanos de Salvador (BA). A aplicação é voltada principalmente para estudantes de **Arquitetura e Urbanismo**, oferecendo embasamento técnico para propostas e intervenções urbanas a partir de dados reais da cidade.

## Objetivo

Promover uma abordagem baseada em evidências no processo de projeto arquitetônico, integrando tecnologia, urbanismo e tomada de decisão informada. A plataforma permite que estudantes:

- Investiguem diferentes bairros de Salvador
- Compreendam perfis populacionais e demográficos
- Identifiquem a distribuição de serviços e equipamentos urbanos
- Fundamentem propostas com dados técnicos e analíticos

## Tecnologias

| Tecnologia | Uso |
|---|---|
| [Streamlit](https://streamlit.io/) | Interface web interativa |
| [Pandas](https://pandas.pydata.org/) | Manipulação e filtragem de dados |
| [NumPy](https://numpy.org/) | Cálculos numéricos |
| [PyDeck](https://deckgl.readthedocs.io/) | Visualização de mapas |
| [Plotly](https://plotly.com/) | Gráficos interativos |
| [Matplotlib](https://matplotlib.org/) + [Seaborn](https://seaborn.pydata.org/) | Gráficos estáticos |

## Páginas da Aplicação

| Página | Descrição |
|---|---|
| Página Inicial | Apresentação da plataforma e seus objetivos |
| Dados Sócio-Demográficos | Análise de população, densidade e perfil dos bairros |
| Equipamentos Urbanos | Mapeamento de infraestrutura e serviços urbanos |
